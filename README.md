# アプリケーションを作成した背景
自分の参加しているだんじり祭は、本場岸和田のだんじりに比べると知名度はなく観客も地域の方のみとなっています。
他の地域の方などにもっと認知してもらうためには、楽しさなどを発信する必要があると考えました。
そこで、インターネットを通じて知ってもらうためにホームページ作成をすることを決めました。
